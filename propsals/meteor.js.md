# meteor.js
## Philippe Lewicki
### Phil is on [github](https://github.com/philfree) and [twitter](https://twitter.com/philfree)

Meteor is a pure javascript open source platform based on node.js, mongodb, mustache style template with optional  packages like backbone, underscore, bootstrap, coffeescript, less, stylus, d3, jquery...
I will present the key concept like live page update, synchronization, hot code push then demo some application and explain how some of the key concept are implemented.
